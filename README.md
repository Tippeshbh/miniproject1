# Student Report Card System (Mini Project)

This project is a simple **Student Report Card System** built using **Python and Object-Oriented Programming (OOP)** concepts.

It allows adding student marks, calculating averages, checking pass/fail status, and generating a report card.

---

## Features

- Add student details
- Add subject marks
- Calculate average marks
- Check pass or fail
- Calculate grades
- Generate report card

---

## OOP Concepts Used

### 1. Encapsulation
- Marks are stored in a private variable `__marks`

### 2. Classes and Objects
- Student class stores student information
- ReportCard class generates report
- ClassRoom class manages students

### 3. Methods
- Add marks
- Calculate average
- Generate report

---

## Classes

### Student
- Add subject marks
- Calculate average
- Check pass or fail
- Calculate grade

### ReportCard
- Generates student report

### ClassRoom
- Stores student list
